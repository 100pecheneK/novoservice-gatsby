# Дипломный проект

Проект размещен по адресу https://novoservice.netlify.app

Функционал Папап принтер:
- график работы
- контакты
- услуги

Функционал Фотоновик:
- график работы
- контакты
- услуги
- **редактор макета и оплата**

# Принцип работы

Добавление всего контента (включая раздела с выбором сервиса) происходит через contentful.
Данные подтягиваются через Graphql.
Используется gatsby для статической генерации после изменения в contentful.
